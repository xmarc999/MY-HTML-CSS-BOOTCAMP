# html element attributes & values
html elements can have attributes and values
example: 
```
<div id="cowgirl"></div>
```
## id
only one per page
in css
```
#cowgirl
```

example:
```
#cowgirl {
    height: 2px;
    border-bottom: 2px solid red;
}
```

## class
as many as you want per page
in css
```
.cowboy
```
